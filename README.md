<h2>개요</h2>
<li>JSP&Servlet을 이용하여 웹 쇼핑몰 프로젝트를 진행하며, 
이후 SpringFrameWork 환경에서 프로젝트를 다시 개발</li>
<li>쇼핑몰에서 제공하는 물품등록,구매,리뷰,마이페이지,(관리자)페이지를 개발한다.</li>
